## User authentication with JWT refresh tokens

Created Full-stack web app using REST-api, JWT tokens and MySQL as backend. Reactjs as front end.

## Libraries used

Frontend:(vite)react, axios, react-router-dom 
Api: express, bcrypt, dotenv, mysql2, cookie-parser, cors, jsonwebtoken, nodemon
